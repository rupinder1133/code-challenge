Problem 1


Problem Statement

Three ways to sum to n

Approach 1

O(1) best performance 

var sum_to_n_a = function(n) {
    return n * (n + 1) / 2;
};

Approach 2

O(n) brute force 

var sum_to_n_b = function(n) {
    let sum = 0;
    for (let i = 1; i <= n; i++) {
        sum += i;
    }
    return sum;
};

Approach 3

O(n) functional style - recursion 

var sum_to_n_c = function(n) {
    if (n <= 0) return 0;
    return n + sum_to_n_c(n - 1);
};
